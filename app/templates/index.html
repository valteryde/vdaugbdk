fondi Visit the code

Helper tool for rendering LaTeX math equations written purely in Python

Spinning up a full latex process is redundant for simple equations. Fondi fixes this by providing a ultra lightweight alternative.

 

import fondi
text = fondi.MathText("f(x) = x^2 - 2")
text.image.save("equation.png")
